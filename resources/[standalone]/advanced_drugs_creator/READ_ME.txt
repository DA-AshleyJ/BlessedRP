ESX Installation:
	1. Drop the script in your resources
	2. Add the script automatic start in server.cfg
	3. When the script starts, it should automatically setup the database
	
	Optional
	4. If you want to use the premade drugs, you can manually execute the files in "advanced_drugs_creator/sql/******_data.sql" and "advanced_drugs_creator/sql/items_limit.sql" OR "advanced_drugs_creator/sql/items_weight.sql" in your database
	
QB-Core Installation:
	1. Drop the script in your resources
	2. Add the script automatic start in server.cfg
	3. Follow this few steps to install the menu and the items https://jaksam1074.github.io/Drugs-Creator-documentation/qb-core_installation/
	4. When the script starts, it should automatically setup the database
	
	Optional
	5. If you want to use the premade drugs, you can manually execute the files in "advanced_drugs_creator/sql/******_data.sql" in your database
