Benvenuti nella breve introduzione a ChocoHax
Aggiornata il: 10/02/2022

1. Dopo aver scaricato l' archivio ZIP, creare una nuova cartella nella cartella `resources` , potrete chiamarla come volete, NON utilizzate numeri o caratteri speciali
2. Estrarre tutti i file nella cartella appena creata
3. Creare un file config.json dal pannello da dove avete scaricato l' aggiornamento
4. Spostare/Copiare il file config.json nella cartella di ChocoHax (dov'e anche presente il file fxmanifest.lua)
5. Avviare il server e ChocoHax

Non appena avrete avviato il server il file config.json e il file permissions.json verrano caricati

Se dovete configurare ChocoHax fatelo utilizzando il pannello dedicato oppure editando manualmente il file config.json
NON toccare i file al interno della cartella `internal`, potrebbe corrompere il funzionamento corretto del anticheat

Potrete usare i comandi `lynx` e `lynx help` per avere maggiori informazioni sui comandi disponibili

Per settare i permessi consultare la guida online aggiornata al link: https://forum.lynxcollective.ltd/knowledge-base/

Per problemi o bug: https://forum.lynxcollective.ltd/forums/support.12/ oppure contattando il nostro supporto
