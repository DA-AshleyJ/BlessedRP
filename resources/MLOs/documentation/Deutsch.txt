Willkommen zur kurzen Einführung in ChocoHax
Aktualisiert am: 02.10.2022

1. Erstellen Sie nach dem Herunterladen des ZIP-Archivs einen neuen Ordner im Ordner "Ressourcen". Sie können ihn benennen, wie Sie möchten, verwenden Sie KEINE Zahlen oder Sonderzeichen
2. Extrahieren Sie alle Dateien in den neu erstellten Ordner
3. Erstellen Sie eine config.json-Datei aus dem Bereich, in dem Sie das Update heruntergeladen haben
4. Verschieben / kopieren Sie die Datei config.json in den Ordner ChocoHax (wo auch die Datei fxmanifest.lua vorhanden ist)
5. Starten Sie den Server und ChocoHax

Sobald Sie den Server gestartet haben, werden die Datei config.json und die Datei permissions.json geladen

Wenn Sie ChocoHax konfigurieren müssen, tun Sie dies über das dedizierte Bedienfeld oder durch manuelles Bearbeiten der Datei config.json
Berühren Sie NICHT die Dateien im „internen“ Ordner, da dies die ordnungsgemäße Funktion des Anticheats beeinträchtigen kann

Sie können die Befehle „lynx“ und „lynx help“ verwenden, um weitere Informationen zu den verfügbaren Befehlen zu erhalten

Informationen zum Festlegen der Berechtigungen finden Sie im aktualisierten Online-Leitfaden unter folgendem Link: https://forum.lynxcollective.ltd/knowledge-base/

Bei Problemen oder Fehlern: https://forum.lynxcollective.ltd/forums/support.12/ oder kontaktieren Sie unseren Support
